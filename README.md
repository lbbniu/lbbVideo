# lbbVideo
CC视频apicloud定制模块
##ios
open  打开视频界面
start 开始播放
stop  暂停播放
close 关闭视频界面
download 下载视频，默认直接开始下载
downloadStart 开始下载
downloadStop  停止下载
rmVideo 删除视频下载
init app从后台进入前台，需要调用此方法启动解密服务
##android
open  打开视频界面
start 开始播放
stop  暂停播放
close 关闭视频界面
download 下载视频，默认直接开始下载
downloadStart 开始下载
downloadStop  停止下载
rmVideo 删除视频下载
